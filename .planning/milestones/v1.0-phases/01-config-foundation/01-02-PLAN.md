**GREEN 阶段:** 创建 `profile_manager.py`，实现 ProfileManager 类：

1. `__init__(self, base_dir: Path | None = None)`: 接受可选的 base_dir 参数，默认 `~/.chatgpt-register/profiles/`。使用 `expanduser().resolve()` 处理路径。

2. `save(self, name: str, config: RegisterConfig) -> Path`:
   - 校验 name（仅允许 `[a-z0-9][a-z0-9_-]*`，最长 64 字符，不含路径分隔符）
   - 文件路径: `self.base_dir / f"{name}.toml"`
   - 创建目录: `path.parent.mkdir(parents=True, exist_ok=True)`
   - 序列化: `config.model_dump(mode="json", exclude_none=True)`
   - 写入: `tomli_w.dump(data, f)` 以 `"wb"` 模式
   - 返回文件路径

3. `load(self, name: str) -> RegisterConfig`:
   - 检查文件存在，不存在则 `raise FileNotFoundError(f"Profile 不存在: {path}")`
   - 读取: `tomllib.load(f)` 以 `"rb"` 模式
   - 反序列化: `RegisterConfig.model_validate(data)`

4. `list_profiles(self) -> list[str]`:
   - 目录不存在返回空列表
   - `sorted(p.stem for p in self.base_dir.glob("*.toml"))`

5. `exists(self, name: str) -> bool`: 检查文件是否存在

6. `delete(self, name: str) -> None`: 存在则删除，不存在不报错
