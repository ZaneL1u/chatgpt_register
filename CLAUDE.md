# 项目语言约束

本仓库默认要求使用简体中文输出。

## 强制要求

- 所有 GSD 工作流产物、阶段讨论、计划、验证报告、进度说明、执行总结，默认全部使用简体中文。
- 所有 Codex 直接回复用户的说明、分析、总结、代码评审、变更说明，默认全部使用简体中文。
- 新增 `.planning/` 下的文档时，除非用户明确要求其他语言，否则统一使用简体中文。

## 允许保留原文的内容

- 代码、命令、路径、环境变量、接口字段、协议名、日志片段、第三方产品名、报错原文，可以保留原始语言。
- 用户明确要求英文输出时，只对当前明确指定的那部分内容切换语言，不扩散到其他输出。

## 执行细则

- 说明性文字保持中文，不要为了模板整齐默认切到英文标题或英文段落。
- 必须中英混排时，中文负责解释，英文只保留最小必要技术表面。
- 如果历史模板或上游工作流默认英文，仍以本文件要求为准，优先输出中文版本。
